Bruto Blog

Diseño y desarrollo de un blog de estilo brutalista y minimalista.

Cliente

Proyecto Personal

Servicios

Diseño UI, Desarrollo Front-End

Año

2023

Ver Sitio Web

El Desafío

El objetivo era crear un espacio de lectura online que rompiera con los diseños de blog tradicionales. Se buscaba una estética cruda y directa, inspirada en el brutalismo digital, donde el contenido y la tipografía fueran los únicos protagonistas, garantizando una legibilidad excelente sin distracciones.

Nuestra Solución

Desarrollamos "Bruto Blog" usando HTML y CSS puros, enfocándonos en una estructura semántica y un rendimiento óptimo. Se seleccionó una paleta de colores de alto contraste y una tipografía serif clásica para el cuerpo de texto, creando una base sólida para la lectura. El diseño elimina todos los elementos no esenciales para centrar la atención del usuario en las palabras.
